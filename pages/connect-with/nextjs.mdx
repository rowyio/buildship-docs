# How to Use BuildShip with Next.js (v0 by Vercel)

This tutorial video guides users in building a low-code app by combining the capabilities of v0 by Vercel for the
frontend and BuildShip for the backend. The tutorial covers designing dynamic front-end interfaces with v0, establishing
workflows in BuildShip, making API calls, and integrating the front-end and back-end seamlessly. This video is ideal for
anyone aiming to simplify their app development process using the low-code features of v0 by Vercel and BuildShip.
